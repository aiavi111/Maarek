# MAAREK — Ресторан с панорамой Бишкека

Премиальный сайт-приложение для ресторана Maarek (17 этаж БЦ Россия): чёрно-золотая тема по фирменной монограмме, русский интерфейс, цены в сомах. Next.js 15, React 19, Tailwind CSS v4, Framer Motion.

## Запуск

```bash
npm install
npm run dev
```

Телефон — основной формат; с ≥1024 px раскладка расширяется до 2–4 колонок. Деплой на Vercel без настроек.

## Бренд

Золотая монограмма (два шеврона + ромб) перерисована в SVG — `src/components/logo.tsx`. Акцент `--color-accent: #c9a257` на тёплом чёрном.

## Данные

`src/data/menu.ts` — 24 позиции: завтраки от бренд-шефа, стейки на углях, гриль, бургеры, том-ям, десерты. Халал: без свинины и алкоголя. Адрес и рейтинг — 2ГИС. Промокоды: `MAAREK10`, `PANORAMA15`. Фото — Unsplash, замените на фирменные.
